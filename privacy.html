1. Introduction

This Privacy Policy explains how the Aurora Discord Bot ("Aurora", "the Bot") collects, uses, stores, and protects information when users interact with the Bot on Discord.

By using Aurora, you agree to the collection and use of information in accordance with this policy.

2. Information We Collect

Aurora may collect:

  • Discord User ID
  • Discord Server (Guild) ID
  • Discord Channel ID
  • Command usage data

Aurora does not collect personal data such as names, emails, or passwords.

3. How Information Is Used

  • Providing bot functionality
  • Processing commands
  • Improving performance
  • Preventing abuse
  • Saving server settings

4. Data Storage

Some data may be stored in databases to ensure proper functionality. This may include IDs and configuration data.

5. Data Sharing

Aurora does not sell or share user data with third parties. Data may only be disclosed if required by law.

6. Data Security

Reasonable measures are used to protect data, but no system is 100% secure.

7. Data Removal

Removing the bot from a server stops data processing. Users may request data deletion where applicable.

8. Children's Privacy

Aurora is not intended for users below Discord’s minimum age.

9. Changes to This Policy

This policy may be updated. Continued use means acceptance of changes.

10. Contact

For questions, contact the developer via Discord.
