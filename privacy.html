Aurora Bot – Privacy Policy

Last Updated: March 9, 2026

1. Introduction

This Privacy Policy explains how the Aurora Discord Bot ("Aurora", "the Bot") collects, uses, stores, and protects information when users interact with the Bot on the Discord platform.

By using Aurora, you agree to the collection and use of information in accordance with this Privacy Policy.

2. Information We Collect

Aurora may collect and process the following types of information:

  • Discord User ID
  • Discord Server (Guild) ID
  • Discord Channel ID
  • Command usage information

Aurora does not collect personal information such as real names, email addresses, passwords, or private account data.

3. How Information Is Used

The collected information may be used for the following purposes:

  • Providing bot functionality
  • Processing commands
  • Improving bot performance
  • Preventing abuse or misuse of the bot
  • Maintaining server-specific settings

4. Data Storage

Some data may be temporarily or permanently stored in databases in order to allow Aurora to function properly.

Stored data may include identifiers such as User IDs, Server IDs, or configuration data necessary for bot features.

5. Data Sharing

Aurora does not sell, trade, or share user data with third parties.

Data may only be disclosed if required by law or if necessary to protect the security and integrity of the bot service.

6. Data Security

Reasonable technical measures are taken to protect stored data from unauthorized access, misuse, or loss.

However, no system can guarantee absolute security.

7. Data Removal

Server administrators may remove Aurora from their server at any time, which will stop further data processing related to that server.

Users may also request data removal where applicable.

8. Children's Privacy

Aurora is not intended for use by individuals under the minimum age required to use the Discord platform.

9. Changes to This Policy

This Privacy Policy may be updated periodically to reflect changes in the bot's functionality or legal requirements.

Continued use of Aurora after updates indicates acceptance of the revised policy.

10. Contact

If you have any questions regarding this Privacy Policy, you may contact the developer through Discord.
